# Eurostat Energy Prices Explorer (Streamlit)

## Run locally
1. Put your Excel file next to `streamlit_app.py`
2. Install deps:
   pip install -r requirements.txt
3. Run:
   streamlit run streamlit_app.py

## Deploy (Streamlit Community Cloud)
- Push these files to a GitHub repo:
  - streamlit_app.py
  - requirements.txt
  - Your Excel file (or upload at runtime)
- Create a new Streamlit app from the repo.
